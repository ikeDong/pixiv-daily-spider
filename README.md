# pixiv_id_spider

本项目采用Scrapy框架，对pixiv爬取每日排行榜图片的**标题**，**作者**，**id**，**所含页数**，**排名**。
预计采用django+vue实现网页端图片展示。

**本项目不会下载图片**

开发思路：通过爬取上述的图片数据，存入到数据库中。再使用[pixiv代理](pixiv.re)，通过id及所含页数信息在前端展示图片。
